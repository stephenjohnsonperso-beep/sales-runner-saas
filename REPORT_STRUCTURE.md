# Structure du Rapport d'Intelligence Exécutive - Benchmark

## Format et Présentation
- **Type** : Rapport d'Intelligence Exécutive (14-29 pages)
- **Titre** : "RAPPORT D'INTELLIGENCE EXÉCUTIVE"
- **Sous-titre** : "Analyse Stratégique • Opportunité d'Affaires"
- **Couverture** : Fond bleu foncé, logo entreprise, lieu du projet
- **Classification** : Confidentiel
- **Préparé pour** : [Client Name]

## Structure du Rapport

### Page 1 : Couverture
- Titre principal
- Nom de l'entreprise cible
- Localisation
- Projet/Contexte
- Date de préparation
- Niveau de confidentialité

### Page 2 : Sommaire Exécutif (Executive Summary)

#### L'OCCASION
- Description du projet/opportunité
- Contexte stratégique
- Urgence et timing
- Besoins identifiés

#### APERÇU DE L'ENTREPRISE
**Profil de l'Entreprise** (Tableau)
- Raison sociale
- Siège social
- Site du projet
- Fondation/Âge
- Nombre d'employés (groupe et site)
- Revenus (groupe)
- Bénéfice net
- Propriétaire
- Secteur
- Site web

#### POSITION DE MARCHÉ
- Part de marché
- Trajectoire
- Réputation
- Marques emblématiques

#### DIFFÉRENCIATEURS CLÉS DU PROJET
- Points forts technologiques
- Avantages compétitifs (liste à puces)
- Innovations clés

### Page 3 : Aperçus Clés pour Ventes B2B

#### TABLE : APERÇU STRATÉGIQUE vs IMPLICATION POUR [CLIENT]
| # | Aperçu Stratégique | Implication pour [Client] |
|---|---|---|
| 1 | Description du point stratégique | Impact/Opportunité pour le client |
| ... | ... | ... |

#### ÉVALUATION DU LEAD - BANT+
**Tableau de Notation**
| Critère | Évaluation | Justification |
|---|---|---|
| BUDGET | A | Montant alloué et approbation |
| AUTORITÉ | A | Décideurs identifiés |
| BESOIN | A+ | Besoins spécifiques identifiés |
| TIMING | A+ | Timeline du projet |
| FACTEURS + | A+ | Rentabilité, durabilité, partenariats |

#### VERDICT FINAL
- Score global (ex: QUALIFIÉ GO - PRIORITÉ A+)
- Résumé de la qualification
- Raison de la qualification

#### VALEUR STRATÉGIQUE DE L'OPPORTUNITÉ
**Indicateurs clés** (Cartes de statistiques)
- Investissement total
- Nombre d'employés post-expansion
- Date de mise en service
- Potentiel de ventes

## Éléments de Design
- **Couleurs** : Bleu foncé (#0F172A), accents verts pour les points positifs
- **Typographie** : Sans-serif moderne, hiérarchie claire
- **Sections** : Séparateurs rouges pour les titres principaux
- **Tableaux** : En-têtes bleu foncé, alternance de lignes
- **Mise en évidence** : Boîtes vertes pour les points clés, boîtes bleues pour les profils

## Contenu Clé à Générer
1. Analyse de l'opportunité
2. Profil détaillé de l'entreprise cible
3. Analyse stratégique avec implications pour le client
4. Évaluation BANT+ personnalisée
5. Verdict de qualification
6. Valeur stratégique quantifiée
